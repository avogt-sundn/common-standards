Frontend code formatting and linting standards for Angular projects.

Formatting

  • Formatter: Prettier
  • Print width: 140
  • Indent: 2 spaces
  • Quotes: single
  • Semicolons: always
  • Trailing commas: none

Linting

  • Linter: ESLint with angular-eslint
  • Components: standalone (Angular 17+)
  • Templates: accessibility rules enforced
